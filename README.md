# Evrenzo Banners

Web ad banners for Evrenzo

### Getting Started

```shell
npm install
npm start
```

### Packaging

You will need the following dependencies to generate GIFs:

```shell
brew install convert
```

```shell
npm install
npm run build

# Generate GIFs
./bin/generate-gifs.js
```
